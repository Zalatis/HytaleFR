# Patch FR complet (uniquement Client Side) pour Hytale

Ce dépôt contient les fichiers de traduction communautaire pour passer le client Hytale en Français. Le but est de fournir une traduction complète de l'interface, des objets et des menus pour les francophones.

## Comment installer?

Il suffit de prendre les 3 fichiers .lang contenus dans le rep et les glisser dans %appdata%/Hytale/install/release/package/game/latest/Client/Data/Shared/Language/fr-FR

## Elle contient quoi?

La traduction contient tout les items/menus du jeu (~10000 lignes de traduction)
